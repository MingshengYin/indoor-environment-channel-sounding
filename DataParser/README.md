# P2M parser and data extracter

Parsing the ray tracing data from P2M file and extract to CSV and MAT files.

Download projects in link: https://drive.google.com/drive/folders/17yg5cQhERSmtUhX7pFEe7XgrGz02UPXB?usp=sharing

Put a whole project folder into folder /DraftData, 
for example: /DraftData/'ProjectName'/'ProjectName'.p2m (10 p2m files)

Then the .mat file stores in /matResults, 
for example: /matResults/'ProjectName'/'ProjectName'_Tx_1_data.mat

To run in Matlab, put a "XXX_Tx_X_data.mat" and a "XXX_roomIndex.mat" files in Matlab script folder.
